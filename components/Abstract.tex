\textbf{\large Хураангуй}

Энэхүү судалгааны ажлаар машин сургалтын ансамбль аргуудыг ашиглан валютын зах зээл (Forex) дээрх EUR/USD хослолын үнийн чиг хандлагыг таамаглах, автомат арилжааны дохио үүсгэх систем болон гар утасны аппликейшн хөгжүүлсэн.

Системийн цөм нь LightGBM, XGBoost, CatBoost гэсэн гурван градиент нэмэгдүүлсэн шийдвэрийн модны (GBDT) ансамбль загвар юм. Загваруудыг 2015--2022 оны $\sim$2.97 сая бааранд сургаж, 2023 оны 371,000 бааранд баталгаажуулж, 2024 оны 371,000 бааранд тестэлсэн. Walk-forward validation аргачлалыг хэрэглэж overfitting-ийн эрсдэлийг бууруулсан. 6 хугацааны интервалаас (M1--H4) нийт 48 техник шинж чанар тооцоолж, олон хугацааны хүрээний дүн шинжилгээг хэрэгжүүлсэн.

2025 оны бэктест дүн: 10,000\$ анхны хөрөнгөөс 14,161.20\$ болж \textbf{+41.61\%} өгөөж, \textbf{Profit Factor 2.46}, \textbf{Sharpe Ratio 9.64}, \textbf{хамгийн их уналт 3.93\%} хүрсэн. React Native технологи дээр суурилсан ``Predictrix'' мобайл аппликейшн нь бодит цагийн ханш, ML дохио, эдийн засгийн мэдээ, хэрэглэгчийн баталгаажуулалт зэрэг бүрэн функцийг хэрэгжүүлсэн.

\begin{center}
\textbf{\large Abstract}
\end{center}

\begin{english}
This research develops an automated trading signal generation system and mobile application for predicting EUR/USD price movements in the Forex market using machine learning ensemble methods.

The core of the system is an ensemble of three Gradient Boosted Decision Tree (GBDT) models: LightGBM (496 trees), XGBoost ($\sim$400 trees), and CatBoost (499 trees). The models were trained on $\sim$2.97 million bars from 2015--2022, validated on 371,000 bars from 2023, and tested on 371,000 bars from 2024. Walk-forward validation methodology was employed to mitigate overfitting risks. A total of 48 technical features were computed from 6 timeframes (M1, M5, M15, M30, H1, H4), implementing multi-timeframe analysis.

Backtest results for 2025: starting from \$10,000 initial capital, the system achieved \$14,161.20 final balance (\textbf{+41.61\%} return), \textbf{Profit Factor 2.46}, \textbf{Sharpe Ratio 9.64}, and \textbf{Maximum Drawdown 3.93\%}. The ``Predictrix'' mobile application, built with React Native, provides real-time exchange rates, ML signals, economic news, and user authentication.

\vspace{3mm}
\textit{Keywords: machine learning, ensemble model, foreign exchange market, GBDT, XGBoost, LightGBM, CatBoost, technical analysis, trading signals, React Native, backtesting}
\end{english}
